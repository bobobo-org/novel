# 諸天萬界小說生成系統

這個 repo 目前是一個離線可用的單頁 HTML 小工具，用來產生小說題材、故事種子、章節草稿、續寫路線與 ChatGPT 接力提示詞。

## 目前內容

- `index.html`：部署首頁，會自動導向主程式。
- `諸天萬界小說生成系統_v4精簡可用離線故事資料庫版_v2完成版/諸天萬界小說生成系統_v4豪華版_主題相容ChatGPT接力版.html`：主要 app。
- `諸天萬界小說生成系統_v4精簡可用離線故事資料庫版_v2完成版/使用說明_README.txt`：原始使用說明。
- `諸天萬界小說生成系統_v4精簡可用離線故事資料庫版_v2完成版/點我開啟.bat`：Windows 本機開啟用批次檔。

## 技術狀態

- 無建置步驟。
- 無後端 API。
- 無 Supabase 資料表或 Edge Function。
- 作品資料儲存在瀏覽器 `localStorage`。

## Vercel

Vercel 可以直接部署此資料夾。`vercel.json` 已設定 `/` 導向主 HTML 檔。
