諸天萬界小說生成系統 v4精簡可用版｜離線故事資料庫擴充版

這一版以 v4豪華版介面為基礎，刪除不能實際使用的部分：直連、金鑰儲存、測試連線、假 AI 模式，以及會讓流程變複雜的總控與診斷功能。

保留可用功能：v4 深色分頁介面、題材庫、主題相容限制、本機建立第一章、本機續寫、十章大綱、ChatGPT 接力提示詞、複製提示詞、複製全文、複製設定、TXT 匯出。

本次新增「離線故事資料庫」：
每一類題材都內建人物關係、場景事件、章節推進、反轉伏筆、章尾鉤子、A/B/C 選項後果。
所以不接 AI 也能產出比較有故事感的章節，不會只剩設定表。

目前 16 類題材包：
附身變身、都市奇幻、豪門權謀、少女漫畫、娛樂圈、職場商戰、醫療重生、宮廷宅鬥、仙俠修真、玄幻升級、末世求生、科幻未來、懸疑推理、遊戲系統、經營養成、諸天穿梭。

分類邏輯：
現代題材只搭配現代、豪門、娛樂圈、職場、醫療、都市奇幻等元素。
古代題材只搭配宮廷、宅鬥、王朝、攝政王、嫡庶、後宮等元素。
仙俠玄幻只搭配宗門、靈根、修煉、血脈、秘境、天命等元素。
科幻末世只搭配 AI、賽博、星際、基地、感染、資源、生存等元素。
遊戲系統與諸天副本可跨題材，但會透過故事引擎限制，不會亂接不相容背景。

建議流程：
1. 打開 HTML，或點「點我開啟.bat」。
2. 到「題材庫」選主題大類。
3. 系統會自動排除不相容組合。
4. 按「產生故事種子」，會看到人物關係、場景、轉折、伏筆、鉤子資料。
5. 到「建立故事」按「建立第一章」或「產生十章大綱」。
6. 到「續寫」選路線，離線版會用故事資料庫繼續產章節。
7. 如果要用 ChatGPT 算力，到「存檔 / ChatGPT」按「給 ChatGPT：直接開始遊戲」。
8. 按「複製提示詞」，貼到 ChatGPT。

注意：這不是直連版，不需要 API Key。這是「離線可玩 + ChatGPT 接力」版。
