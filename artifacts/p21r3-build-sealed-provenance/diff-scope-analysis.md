# Diff Scope

Included:

- build-time provenance generation and validation
- shared release metadata contract
- runtime health provenance verification
- static legacy release stamping from the sealed artifact
- focused regression coverage
- RC3 release metadata

Excluded:

- Studio behavior
- Story Bible and canonical data
- IndexedDB and SQLite schemas
- backup and restore behavior
- AI routing
- Vercel environment variables
- Production deployment or alias changes
- immutable RC2 evidence and tags
