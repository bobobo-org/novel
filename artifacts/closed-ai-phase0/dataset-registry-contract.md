# Dataset registry contract

Version: `closed-ai-dataset-registry-v1`

Dataset versions are immutable and include owner scope, lifecycle status, source record IDs, content hash, provenance validation, quality score, contamination flags, deduplication fingerprint, and creation time. Personal datasets require an owner and are inaccessible across users. Deleted or contaminated source material is ineligible.

No real shared dataset is generated in Phase 0.
