# Benchmark contract

Version: `closed-ai-benchmark-v1`

The benchmark separates:

1. `contract_test`
2. `deterministic_regression`
3. `model_quality_evaluation`
4. `human_review`
5. `production_e2e`

Phase 0 executes only contract tests and deterministic regression. Model quality, human review, and Production E2E remain unexecuted interfaces and must not be presented as passed model capability.
